# SIT780
Community markerplace:
a website for small retailers to sell products online then customers nearby can purchase goods online without any physical touch.
--------------------------------------------------------------------------------

From a command/shell terminal
* `cd` into the parent directory you want to install the project in
* `git clone` the project into a child directory
* `cd` into that child directory
* run `npm install` to install dependencies

For example:

    $ cd Projects
    $ git clone https://github.com/Qiaoliwang2020/community-marketplace-nodejs.git


        ... git output here ...

    $ cd community-marketplace-nodejs

    $ npm install

        ... npm output here ...


run locally
--------------------------------------------------------------------------------

After installing, run the server using

    npm run dev

This should print the following to the console.

    community-marketplace-nodejs: server starting on http://localhost:8080
